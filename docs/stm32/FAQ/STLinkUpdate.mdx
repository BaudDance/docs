# 升级 STLINK 固件

通过 cubeIDE 升级 ST-LINK 下载器的固件

## 📌 为什么要升级？

ST 官方会不定期推送 STLINK 的固件更新

有新版本时就会弹出升级提示，建议及时升级

<img src={require('./img/STLinkUpdate/linkNewVersion.png').default} alt="升级提示" width="460px" />
</div>

## 📌 升级方法

#### 1️⃣ 在软件顶部的菜单栏，按图片指示打开升级工具

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/STLinkUpdate/findupgtool.png').default} alt="ST-LINK升级页面" width="450px" style={{boxShadow: "0px 0px 5PX 2px rgba(0, 0, 0, 0.1), inset 0 0 50px rgba(0, 0, 0, 0.3)"}}/>
</div>

#### 2️⃣ 拔掉ST-LINK，等3秒再插上
:::warning
**此时不要关掉第 1 步的页面**
:::
<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/STLinkUpdate/拔出STLINK.png').default} alt="拔出STLINK" width="220px" />
</div>

#### 3️⃣ 按图操作，依次点击1-2-3按钮（第二步的按钮需要点击 2 次）

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/STLinkUpdate/step_upg_stlink.png').default} alt="操作步骤" width="550px" />
</div>

#### 4️⃣ 进度条走完后，等5秒关掉升级界面，升级完成😎

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/STLinkUpdate/upgfinished.png').default} alt="upgfinished" width="550px" />
</div>


## 📌 升级遇到问题?

- 提示 `ST-Link device lost when exiting upgrade mode`

    只要升级进度条走完就可以了，可以忽略这个提示

- 即使重新插拔ST-LINK，仍然找不到设备，按钮为灰色

    尝试换个USB接口，重新上述步骤

- 提示：通过反馈发现，市面存在部分ST-LINK无法升级，原因不明，建议换个ST-LINK

    > 当然，我们会推荐使用波特律动学习套件😎
