# 手动安装固件包

由于ST服务器在国外，部分网络会打不开。可以通过手动下载固件包进行安装。

- **下载固件包：**<a href="https://pan.baidu.com/s/1MTFNos5LCT9sLLxVn4mqgA?pwd=6666" class="source_download_button">点击下载</a>

    > **注意：文件夹内的两个zip都要下载**

- **打开 preferences**

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('../img/manualInstallFirmware/preferences.png').default} alt="NTC线" width="450px" />
</div>

- **在 STM32Cube -> Firmware Updater 中找到库存放路径**

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('../img/manualInstallFirmware/FirmwareUpdater.png').default} alt="NTC线" width="650px" />
</div>

- **将下载的固件包 `stm32cube_fw_f1_v180` 放到该路径下** 

    > **注意：不要解压缩**

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('../img/manualInstallFirmware/v180.png').default} alt="NTC线" width="550px" />
</div>

- **打开 Manage Embed Software Packages**

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('../img/manualInstallFirmware/ManagePackages.png').default} alt="NTC线" width="450px" />
</div>

- **点击 From Local **

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('../img/manualInstallFirmware/fromLocal.png').default} alt="NTC线" width="550px" />
</div>

- **选择 `stm32cube_fw_f1_v185`，然后点击 打开**

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('../img/manualInstallFirmware/v185.png').default} alt="NTC线" width="550px" />
</div>

- **等待导入完成后，即可创建工程了**

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('../img/manualInstallFirmware/newProj.png').default} alt="NTC线" width="650px" />
</div>
