# 如何查找我遇到的问题

首先，看看在以下哪一步中遇到了问题？

## 📌 我在安装 cubeIDE 时遇到问题

- 我的安装包打不开：[**解决方法🔗**](docs/stm32/FAQ/InstallError.mdx)

- 我不能登录 ST 账号

    - 可以查看详细登录方法：[**详细登录方法🔗**](docs/stm32/FAQ/login.mdx)

    - 我的电脑不能联网：[**绕过登录，手动安装芯片包🔗**](docs\stm32\FAQ\manualInstall.mdx)

#### 6️⃣ 没有可下载工程

因为下载按钮是从历史记录找“上次下载的工程”

新工程有可能找不到，需要右键工程，选择 `Run as` 下载（如图）

如此操作一次，后面即可正常
