# 缺少 STLINK Server 组件

## 📌 什么是 STLink Server？

STLink Server（下文简称“服务”） 是 CubeIDE 软件访问 STLink 的桥梁

下载程序时，CubeIDE 会连接到服务，而非直接连接到 ST-LINK 物理接口（USB）

因此，当 STLink Server（服务）未安装时，CubeIDE 会弹窗出现如下报错：
