# 缺少 STLINK Server 组件

## 📌 什么是 STLink Server？

STLink Server（下文简称“服务”） 是 CubeIDE 软件访问 STLink 的桥梁

下载程序时，CubeIDE 会连接到服务，而非直接连接到 ST-LINK 物理接口（USB）

因此，当 STLink Server（服务）未安装时，CubeIDE 会弹窗出现如下报错：

<img src={require('./img/DownloadError/noDebugSession.png').default} alt="noTitle" width="500px" />
</div>

## 📌 安装 STLink Server

1️⃣ 下载 STLink Server 安装包
<div style={{marginBottom: "20px", marginLeft:"25px"}}>
<a href="https://pan.baud-dance.com/d/ST%E5%B7%A5%E5%85%B7/STLINK_Server.zip" class="source_download_button" target="_blank">立即下载（Server）</a>
</div>

2️⃣ 解压缩到非中文路径下

3️⃣ 双击 **`st-stlink-server.2.1.1-1.msi`** 进行安装即可

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/installServer.png').default} alt="noTitle" width="700px" style={{boxShadow: "0px 0px 5PX 2px rgba(0, 0, 0, 0.1), inset 0 0 50px rgba(0, 0, 0, 0.3)"}}/>
</div>

:::tip
msi 安装包仅适配 Windows 系统，如果你使用苹果电脑，请安装 `.pkg` 文件
:::
