# ST-LINK 初始化出错

**GDB端口被占用**

解决方法：

- **1、打开 Run Configuration**

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('../img/CompilationFailed/runConfig.png').default} alt="runConfig" width="350px" />
</div>

- **2、将端口号改为 31234**

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('../img/CompilationFailed/改端口号.png').default} alt="改端口号" width="600px" />
</div>

- **3、点击 Apply，再点 Run，就能下载程序了**

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('../img/CompilationFailed/点Apply.png').default} alt="点Apply" width="600px" />
