# 安装失败报错

双击安装包时弹出错误窗口：

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px", marginBottom: "40px"}}>
<img src={require('./img/IdeInstallationWizard/安装失败报错.png').default} alt="安装报错" width="400px" />
</div>

## 📌问题1： 安装包在中文路径中

检查安装包所在的文件夹路径中是否存在中文

若有，将安装包移到无中文路径的文件夹下，例如：

| **可以 ✅**       | **不行 ❌**           |
|--------------------|------------------------|
| `D:\Downloads`     | `D:\单片机学习`        |
| `D:\STM32`         | `D:\STM32资料`     |
| ...                | ...                    |

## 📌问题2： 环境变量TEMP中有中文

#### 1️⃣ 点击开始菜单，输入“环境变量”搜索，进入系统属性设置

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px", marginBottom: "40px"}}>
<img src={require('./img/搜索环境变量.png').default} alt="搜索环境变量" width="500px" />
</div>

#### 2️⃣ 点击系统属性下方的“环境变量”，进入环境变量配置页面

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px", marginBottom: "40px"}}>
<img src={require('./img/IdeInstallationWizard/打开环境变量.png').default} alt="打开环境变量" width="500px" />
</div>

#### 3️⃣ 双击 **`TEMP`** 弹出修改框

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px", marginBottom: "40px"}}>
<img src={require('./img/IdeInstallationWizard/env1.png').default} alt="修改环境变量" width="500px" />
</div>

#### 4️⃣ 在修改框里输入  **`%SystemRoot%\TEMP`**  
:::tip
**为避全角符号问题，请直接复制粘贴进去**。注意复制完整，尤其不要漏掉百分号%
:::

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px", marginBottom: "40px"}}>
<img src={require('./img/IdeInstallationWizard/env2.png').default} alt="修改环境变量" width="500px" />
</div>

#### 5️⃣ 用同样的操作方法，修改完这 4 行

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px", marginBottom: "40px"}}>
<img src={require('./img/IdeInstallationWizard/env3.png').default} alt="修改环境变量" width="500px" />
</div>


#### 6️⃣ 修改完成后，点「确定」。然后重新打开IDE安装包，重新安装即可
:::warning
**新建工程、选择工作空间等步骤，也必须确保路径不含中文**。 否则还会出现错误
:::
