# 什么是工作空间

- 工作空间就是一个文件夹📂，里面存放你将要创建的工程。

- 例如：可以选择 `D:\STM32` 为工作空间，以后你新建的工程都会在这个文件夹里。

- 简单来说，工作空间就是你工作的地方，把所有东西放在一起，方便你找到和管理。

- **⚠️注意**： 工作空间路径中**不要有中文**，否则会导致一些问题。
