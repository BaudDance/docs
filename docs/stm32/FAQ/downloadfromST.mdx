# 从ST官网下载安装包

#### 1️⃣ 进入[CubeIDE的下载界面](https://www.st.com/en/development-tools/stm32cubeide.html), 点击获取软件

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px", marginBottom: "40px"}}>
<img src={require('./img/downloadfromST/点击获取软件.png').default} alt="配置工程" width="700px" />
</div>

#### 2️⃣选择对应的版本进行下载（Windows电脑点击最后一个下载即可）

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px", marginBottom: "40px"}}>
<img src={require('./img/downloadfromST/下载软件.png').default} alt="配置工程" width="700px" />
</div>

####  3️⃣ 接下来会弹出一个许可协议，点击接受即可

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px", marginBottom: "40px"}}>
<img src={require('./img/downloadfromST/点击接受.png').default} alt="配置工程" width="700px" />
</div>

#### 4️⃣ 进入登录界面

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px", marginBottom: "40px"}}>
<img src={require('./img/downloadfromST/登录ST账号或者创建自己的ST账号.png').default} alt="配置工程" width="700px" />
</div>

可以点击Create a MyST Account自行注册账号，[点击此处查看注册流程。](/docs/stm32/FAQ/RegistrationProblem)

也可以选择我们准备好的账号密码点击Log in to MyST直接登录。

**账号2**：`st@baud.dance`

**密码2**：`ABC@123@abc` <a className=' cursor-pointer'  onClick={() => { navigator.clipboard.writeText('ABC@123@abc');    alert('密码已复制！');  }}>复制密码</a>

:::warning 
**一定要复制密码！手打输入容易锁号**

若提醒输错密码超过5次，则说明账号被封，换另一组账号密码进行登录即可

（可以帮忙点一下“忘记密码”，up收到邮件后会解锁账号）
:::

#### 5️⃣登录完成后完成下载

最后输入密码点击登录后，等待一会就开始下载啦~ （可能等待时间会长一点）
