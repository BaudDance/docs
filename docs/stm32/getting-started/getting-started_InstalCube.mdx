# 安装 CubeIDE


#### 1️⃣ 首先下载 CubeIDE 安装包

   - 从 [**资料包🔗**](https://www.bilibili.com/video/BV1HM411b78E) 直接下载（推荐）

   - 从 [**ST官网🔗**](https://www.st.com.cn/zh/development-tools/stm32cubeide.html) 下载

> 提示：输入网址：**https://docs.baud-dance.com** 即可用电脑打开本页面

#### 2️⃣ 安装包文件必须放在【非中文】路径下

- 例如：

   | **可以✅**       | **不行❌**           |
   |--------------------|------------------------|
   | `D:\Downloads`     | `D:\单片机学习`        |
   | `D:\STM32`         | `D:\STM32资料`     |

#### 3️⃣ 安装

   - 一路点击 `Next`、`I Agree`、`Install`

   - 出现报错？ [**报错解决方法🔗**](docs/stm32/FAQ/InstallError.mdx)

#### 4️⃣ 打开 CubeIDE，选择工作空间

   - 工作空间也必须选择【**非中文**】路径，例如 `D:\STM32`

   - 进一步了解：[**什么是工作空间？🔗**](docs/stm32/FAQ/whatIsWorkSpace.mdx)

   <div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
   <img src={require('/docs/stm32/getting-started/img/selworkspace.png').default} alt="title" width="500px" />
   </div>

#### 5️⃣ 登录 ST 账号（必要步骤） 

   - ⚠️必要步骤！不登录将无法下载芯片包

   - 请按教程操作： [**如何登录 ST 账号🔗**](docs\stm32\FAQ\login.mdx)
