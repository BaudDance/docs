# 安装 CubeIDE

#### 1️⃣ 首先下载 CubeIDE 安装包

<a href="https://pan.baud-dance.com/d/STM32CubeIDE/st-stm32cubeide_1.19.0_25607_20250703_0907_x86_64.exe" class="source_download_button" target="_blank">立即下载</a>
<div style={{marginBottom: "20px"}}>
</div>

:::tip
安装包适配 Windows 系统，如果你使用苹果电脑，需要获取
 <a href="https://pan.baidu.com/s/1DX3xAUFVLozOQG5PfS2zcg?pwd=6666" target="_blank">**苹果电脑dmg版**</a>
:::
:::warning
**重要！**
安装包文件必须放在「**非中文**」路径下，否则无法打开。

| **可以✅**       | **不行❌**           |
|--------------------|------------------------|
| `D:\Downloads`     | `D:\单片机学习`        |
| `D:\STM32`         | `D:\STM32资料`     |

:::

#### 2️⃣ 安装

一路点击 `Next`、`I Agree`、`Install` 即可。如果中途弹出提示，请选择 `Yes`。
:::tip
出现报错？ <a href="/docs/stm32/FAQ/InstallError" target="_blank">**查看解决方法🔗**</a>
:::

#### 3️⃣ 打开 CubeIDE，选择工作空间

:::warning
工作空间也必须选择「**非中文**」路径，例如 `D:\STM32`。可以进一步了解<a href="/docs/stm32/FAQ/whatIsWorkSpace" target="_blank">**什么是工作空间？🔗**</a>
:::

   <div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
   <img src={require('/docs/stm32/getting-started/img/selworkspace.png').default} alt="title" width="500px" />
   </div>

#### 4️⃣ 登录 ST 账号（必要步骤） 

按教程操作 <a href="\docs\stm32\FAQ\login#-如何登录" target="_blank">**如何登录 ST 账号🔗**</a>
:::warning
**必要步骤**！不登录将无法下载芯片包
:::
