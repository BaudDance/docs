## 升级STLINK固件

由于 ST 官方不定期推送 STLINK 下载器的固件更新

所以新购买的 STLINK 需要先升级固件

#### 1️⃣ 在软件顶部的菜单栏，按图片指示打开升级工具

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginLeft:"20px", marginBottom: "20px"}}>
<img src={require('../FAQ/img/STLinkUpdate/findupgtool.png').default} alt="ST-LINK升级页面" width="450px" style={{boxShadow: "0px 0px 5PX 2px rgba(0, 0, 0, 0.1), inset 0 0 50px rgba(0, 0, 0, 0.3)"}}/>
</div>

#### 2️⃣ 拔掉ST-LINK，等3秒再插上

:::warning
**注意：此时不要关掉第 1 步的页面**
:::
<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginLeft:"20px", marginBottom: "20px"}}>
<img src={require('../FAQ/img/STLinkUpdate/拔出STLINK.png').default} alt="拔出STLINK" width="220px" />
</div>

#### 3️⃣ 按图操作，依次点击1-2-3按钮

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginLeft:"20px", marginBottom: "20px"}}>
<img src={require('../FAQ/img/STLinkUpdate/操作步骤.png').default} alt="操作步骤" width="550px" />
</div>

#### 4️⃣ 进度条走完后，等5秒关掉升级界面，升级完成😎

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginLeft:"20px", marginBottom: "20px"}}>
<img src={require('../FAQ/img/STLinkUpdate/upgfinished.png').default} alt="upgfinished" width="550px" />
</div>
