# 登录STM32CubeMX

CubeMX需要登录才能正常自动下载芯片包，进行代码生成，可按照如下步骤登录：

**一、点上方的Help -> Connection & Updates**
<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/登录1.png').default} alt="登录1" width="400px" />
</div>
**二、切换到myST标签页，点击“Enter myST account information”**
<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/登录2.png').default} alt="登录2" width="400px" />
</div>
**三、在弹出的界面中输入账号密码进行登录**

可以点击右下角的“Create account”按钮注册一个新账号, 或者使用我们提供的账号登录：

**账号1**：`baud-dance.com@outlook.com`

**密码1**：`tK6D6abui#@GnM` <a className=' cursor-pointer'  onClick={() => { navigator.clipboard.writeText('tK6D6abui#@GnM');    alert('密码已复制！');  }}>复制密码</a>

**账号2**：`st@baud.dance`

**密码2**：`ABC@123@abc` <a className=' cursor-pointer'  onClick={() => { navigator.clipboard.writeText('ABC@123@abc');    alert('密码已复制！');  }}>复制密码</a>

:::warning 
**一定要复制密码！手打输入容易锁号**

若提醒输错密码超过5次，则说明账号被封，换另一组账号密码进行登录即可

（可以帮忙点一下“忘记密码”，up收到邮件后会解锁账号）
:::

<img src={require('./img/登录3.png').default} alt="登录3" width="400px" />
</div>
:::tip 

由于网络问题，有些小伙伴有可能无法进入登录页面，此时可以尝试 <a href="/docs/stm32/FAQ/fubemxInstallPackage">**使用FubeMX安装固件包** 🔗</a>

:::
