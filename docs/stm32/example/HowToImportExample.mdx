# 【如何下载例程】

- 在例程页面，点击【下载代码】，获取源码

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/IMPORT_EXAMPLE/Download_button.png').default} alt="gpio config" width="500px" />
</div>

- 解压缩到全英文路径下（如：D:\stm32_workspace），然后双击里面的`.project`文件

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/IMPORT_EXAMPLE/dot_project.png').default} alt="gpio config" width="500px" />
</div>

- 在 CubeIDE 中看见弹窗，点击【OK】，即导入成功

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/IMPORT_EXAMPLE/import_success.png').default} alt="gpio config" width="500px" />
</div>

- 右键刚刚导入的工程，选择【Run As】->【STM32 MCU C/C++ Application】，即可下载到学习板

<div style={{ display: "flex", justifyContent: "center", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/IMPORT_EXAMPLE/run_project.png').default} alt="gpio config" width="500px" />
</div>
