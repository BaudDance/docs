# 如何下载例程

#### 1️⃣ 在例程页面，点击【**下载代码**】，获取源码

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/IMPORT_EXAMPLE/Download_button.png').default} alt="gpio config" width="500px" style={{boxShadow: "0px 0px 6PX 3PX rgba(0, 0, 0, 0.1), inset 0 0 50px rgba(0, 0, 0, 0.3)"}}/>
</div>

#### 2️⃣ 先解压缩，然后双击里面的 **`.project`** 文件

:::warning
**工程存放路径不能包含中文**，否则编译会出错
:::

:::warning
**必须先解压缩**，不能在压缩包里直接双击打开
:::

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/IMPORT_EXAMPLE/dot_project.png').default} alt="gpio config" width="500px" style={{boxShadow: "0px 0px 5PX 3PX rgba(0, 0, 0, 0.1), inset 0 0 50px rgba(0, 0, 0, 0.3)"}}/>
</div>

#### 3️⃣ 出现弹窗，点击 **`OK`**，即导入成功

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/IMPORT_EXAMPLE/import_success.png').default} alt="gpio config" width="500px" />
</div>

#### 4️⃣ 如果出现图中的提示，点击 **`Migrate`**
:::tip
这是因为例程的软件版本较老，点击【Migrate】迁移即可
:::
<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/IMPORT_EXAMPLE/newVersionAvailable.png').default} alt="gpio config" width="450px" />
</div>

#### 5️⃣ 按图示方法下载此工程

【**右键点击**】刚刚导入的工程，选择 **`Run As`**

:::tip
**为什么要这样下载？**
因为我们同时打开多个工程，用此方法能确保下载的是例程，而不是其他工程
:::

</div>
