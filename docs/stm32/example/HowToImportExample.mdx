# 如何下载例程

:::tip
**只需验证功能？ 免导入下载方法👉** 解压后双击「下载这个例程.bat」，即可将例程直接下载到学习板。

如非此情况，请按下列步骤导入：⬇
:::

#### 1️⃣ 下载源码到非中文路径下
:::warning
**源码请勿放中文路径下！ 编译器无法解析中文路径**
:::

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginTop: "10px", marginBottom: "40px"}}>
<img src={require('./img/IMPORT_EXAMPLE/downloadCode.png').default} alt="downloadCode" width="500px" style={{boxShadow: "0px 0px 6PX 3PX rgba(0, 0, 0, 0.1), inset 0 0 50px rgba(0, 0, 0, 0.3)"}}/>
</div>

#### 2️⃣ 先「解压缩」，再双击里面的「.project」文件导入

:::warning
**必须先解压缩！请勿在压缩包里直接打开**
:::

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginTop: "10px", marginBottom: "40px"}}>
<img src={require('./img/IMPORT_EXAMPLE/dot_project.png').default} alt="gpio config" width="500px" style={{boxShadow: "0px 0px 5PX 3PX rgba(0, 0, 0, 0.1), inset 0 0 50px rgba(0, 0, 0, 0.3)"}}/>
</div>

#### 3️⃣ 出现弹窗，点击 **`OK`**，即导入成功

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "40px"}}>
<img src={require('./img/IMPORT_EXAMPLE/importSuccess_cut.png').default} alt="gpio config" width="500px" />
</div>

#### 4️⃣ 如果出现图中的提示，点击 **`Migrate`**
:::tip
这是因为例程的软件版本较老，点击【Migrate】迁移即可
:::
<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginTop: "10px", marginBottom: "40px"}}>
<img src={require('./img/IMPORT_EXAMPLE/newVersionAvailable.png').default} alt="gpio config" width="450px" />
</div>

#### 5️⃣ 按图示方法下载此工程

「**右键点击**」刚刚导入的工程，选择 **`Run As`**

:::tip
**为什么要这样下载？**
因为我们同时打开多个工程，用此方法能确保下载的是例程，而不是其他工程
:::
