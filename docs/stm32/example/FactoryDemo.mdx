# 【综合项目】出厂演示程序

**下载例程代码：** <a href="https://pan.baidu.com/s/1nvJdtG3UE5SBQXWMhcXP9g?pwd=6666" class="source_download_button">下载代码</a> [如何使用例程【点击查看教程】](/docs/stm32/example/HowToImportExample)

学习板出厂时自带的程序。

## 如何使用例程

- 编译并下载程序到学习板，即可看到效果

## 程序效果

- OLED 自动循环显示：温湿度、文字、各种图形、BMP图片

- 按下 KEY1、KEY2、旋转编码器，分别可以控制红灯、绿灯、蓝灯的亮/灭

- 连接TYPE-C接口，在串口助手可以看到串口输出“Hello World!”，发送“OK”可以使蜂鸣器响2声

- 背面 10 个 WS2812 灯珠将自动循环：红色、绿色、蓝色、彩虹特效
