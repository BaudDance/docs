# Keil V5编译器 安装教程


首先，我们需要先完成keil软件的安装：[**点击此处安装🔗**](docs/stm32/keil/installKeil.mdx)

#### 1️⃣ 下载V5编译器安装包并安装

<div style={{marginBottom: "20px", marginLeft:"25px"}}>
<a href="https://pan.baud-dance.com/d/STM32CubeMX/ARMCompiler_506_Windows_x86_b960.zip" class="source_download_button" target="_blank">立即下载</a>
</div>

下载完成后，解压后双击 `setup.exe` 进行安装。

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/安装1.png').default} alt="安装1" width="600px" />
</div>

点击两次 `Next` ，即可到达此界面，点击 `Browse` 选择安装路径。

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/安装2.png').default} alt="安装2" width="600px" />
</div>

此文件必须安装在Keil的ARM文件夹里，所以需要先获取keil的文件路径，可以右键点击快捷方式，选择 `打开文件所在的位置` 。

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/安装3.png').default} alt="安装3" width="200px" />
</div>

根据图中所示，选择并复制 `Keil_v5` 的路径。
<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/安装4.png').default} alt="安装4" width="600px" />
</div>

将路径粘贴到安装路径中,并在后面添加 `ARM\ARM_Compiler_5.06u7\`，点击 `OK`，然后点击 `Next` 继续安装。
<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/安装5.png').default} alt="安装5" width="600px" />
</div>

:::tip
请将红框内的文件路径复制保存好，后续需要用到。
:::

最后一步步点击 `Next`和 `Install`，即可完成安装。

#### 2️⃣ 添加编译器版本

打开keil软件，根据下图所示，打开添加编译器版本的界面。

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/路径1.png').default} alt="路径1" width="600px" />
</div>

此处可以看到，ArmCC版本只有一个V6编译器，点击下方添加其他编译器版本按钮。

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/路径2.png').default} alt="路径2" width="600px" />
</div>



将刚刚安装V5编译器时复制的路径粘贴到红框处，删掉最后的反斜杠 `\`，点击确定即可。

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/路径3.png').default} alt="路径3" width="600px" />
</div>

:::warning
此处的路径需要将复制的 `...\Keil_v5\ARM\ARM_Compiler_5.06u7\` 删掉最后的反斜杠 `\`,只保留 `...\Keil_v5\ARM\ARM_Compiler_5.06u7`，否则会添加失败。

若忘记复制路径，可查看第一步获取Keil路径的方法，得到`...\Keil_v5`后，添加 `\ARM\ARM_Compiler_5.06u7` 即可。
:::

此时就可以看到V5编译器已经添加成功，点击关闭即可。

<div style={{ display: "flex", justifyContent: "left", alignItems: "center", marginBottom: "20px"}}>
<img src={require('./img/路径4.png').default} alt="路径4" width="600px" />
</div>
