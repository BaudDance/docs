# 安装 Keil5

:::tip
**仅当您必须使用 KEIL 时才需要阅读本篇**
:::

:::tip
**如非特殊情况，建议跟随常规教程使用 CubeIDE，无需安装 KEIL**
:::

#### 1️⃣ 下载 Keil5 安装包并双击安装

<a href="https://pan.baud-dance.com/d/STM32CubeMX/mdk542a.exe" class="source_download_button" target="_blank">立即下载</a>
<div style={{marginBottom: "20px"}}>
</div>

:::warning
**重要！**
安装路径必须在「**非中文**」路径下，否则无法运行

| **可以✅**       | **不行❌**           |
|--------------------|------------------------|
| `C:\Keil`     | `D:\单片机学习`        |
| `D:\STM32`         | `D:\工具软件`     |
:::
<div style={{ marginBottom: '20px' }}></div>

#### 2️⃣ 安装 STM32F1 芯片包

<a href="https://pan.baud-dance.com/d/STM32CubeMX/Keil.STM32F1xx_DFP.2.4.1.pack" class="source_download_button" target="_blank">立即下载</a>
<div style={{marginBottom: "20px"}}>
</div>

下载后直接双击安装即可 `STM32F1xx_DFP.2.3.0.pack`

注意安装路径也不能包含中文
<div style={{ marginBottom: '20px' }}></div>

#### 3️⃣ 注册方法（仅限学生使用）

KEIL 是收费软件，安装后默认试用版，只能编译不超过32K的代码

对于学生朋友，无力购买正版的，请参考注册方法：http://www.openedv.com/thread-10334-1-1.html

:::warning
任何商用行为都需要购买正版，或者改用 CubeIDE 免费软件
:::

<div style={{ marginBottom: '40px' }}></div>
